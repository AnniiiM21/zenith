## 🚀 Installation

### Method 1: Load as Developer Extension (Recommended)

1. **Download & Build**

   ```bash
   git clone https://github.com/AnniiiM21/zenith.git
   cd zenith
   npm install
   npm run build-extension
   ```

2. **Install in Chrome**
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (toggle in top-right)
   - Click "Load unpacked"
   - Select the `dist` folder from your Zenith project
   - Done! Look for the Zenith icon in your Chrome toolbar

### Method 2: Chrome Web Store (Coming Soon)

We're working on publishing Zenith to the Chrome Web Store for one-click installation.

## 📖 How to Use

### 🎯 **Basic Timer Usage**

1. **Click the Zenith icon** in your Chrome toolbar to open the popup
2. **Choose a preset**:
   - 🍅 **Pomodoro**: 25min work / 5min break (perfect for study sessions)
   - 🎯 **Deep Work**: 45min work / 15min break (ideal for research)
   - ⚡ **Quick Focus**: 15min work / 5min break (great for quick tasks)
   - 🚀 **Long Session**: 90min work / 20min break (for intensive work)
3. **Click Start** and begin your focused work session
4. **Get notified** when it's time for a break
5. **Review your session** in the Analytics tab

### 📊 **Browser Activity Tracking**

1. **Go to Analytics tab** in the Zenith popup
2. **Click "Start Live Tracking"** before beginning work
3. **Work normally** - Zenith tracks your browsing in the background
4. **Click "Stop & Generate Report"** when your session ends
5. **Review detailed analytics**:
   - Time spent on different websites
   - Tab switching frequency
   - Productivity score
   - Top visited sites

### 💡 **Pro Tips for Maximum Productivity**

**For Students:**

- Start tracking before study sessions to see which websites help vs. distract

- Review analytics weekly to identify study pattern improvements

**For Researchers:**

## 🛠 Development & Building

### Prerequisites

- Node.js 16+
- npm or yarn
- Chrome browser

### Build Commands

```bash
# Install dependencies
npm install

# Build extension for Chrome
npm run build-extension

# Development mode (for Electron version - legacy)
npm run electron-dev

# Build Electron app (legacy)
npm run electron
```

### Extension Development

1. **Make changes** to source files in `src/` or extension files
2. **Rebuild** with `npm run build-extension`
3. **Reload extension** in Chrome extensions page (Developer mode)
4. **Test changes** by clicking the Zenith icon

### File Structure

```
zenith/
├── src/                 # Original React source (used for development)
├── extension/           # Chrome extension source files
│   ├── manifest.json    # Extension configuration
│   ├── background.js    # Service worker (timer & tracking logic)
│   ├── popup.html       # Extension popup UI
│   ├── popup.js         # Popup functionality
│   └── content.js       # Website tracking script
├── dist/               # Built extension files (load this in Chrome)
└── electron/           # Electron version (legacy)
```
